# References

This project studied public ideas from:

- Nuwa Skill: parallel public-person research, cognitive framework extraction,
  expression DNA, evidence transparency, and quality validation.
- MiroFish: branch/world-event thinking and multi-agent simulation framing.
- MCP Python SDK v1.x: FastMCP tools, resources, prompts, stdio server usage,
  and local testing guidance.
- OpenAI Codex manual: current Skill locations and MCP `config.toml` structure.

No code, prompt templates, or proprietary structures were copied from Nuwa,
MiroFish, OASIS, or AGPL-licensed implementations. Persona Continuum uses an
original data model and local SQLite implementation.
