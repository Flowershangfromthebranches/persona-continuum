# Domain Model

Personas contain a versioned manifest, identity, cognition, affect, expression,
relationships, evidence, continuation, evaluation, and runtime state.

Important records include `PersonaManifest`, `EvidenceSource`, `EvidenceClaim`,
`MemoryRecord`, `AffectState`, `NeedState`, `RelationshipState`,
`SessionRecord`, `PreparedTurn`, `ContinuationTask`, and `ContinuationBranch`.

Source kinds distinguish historical facts, self reports, third-party reports,
inference, counterfactual simulation, digital experience, user correction, and
system summary.
